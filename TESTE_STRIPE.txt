Pronto para teste local.

1. Abra o arquivo .env e substitua:
   STRIPE_WEBHOOK_SECRET="COLE_AQUI_O_WHSEC_DO_STRIPE_LISTEN"

2. Rode no terminal:
   stripe listen --forward-to localhost:3000/api/stripe/webhook

3. Copie o whsec_... que aparecer e cole no .env.

4. Rode:
   npm install
   npx prisma generate
   npm run dev

5. Teste a compra.
